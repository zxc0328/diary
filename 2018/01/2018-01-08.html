《忍者秘籍》第二版笔记 第九章

Array

客观的说，这本书对于Array的讲述比较照本宣科。有一些关键的内容没有涉及：

  • YDKJS中对于稀疏数组这个大坑做了详细的介绍。
  • 在对于Array方法的介绍和分类（immutable和mutable）上，Learning JavaScript要更出色一些。
  • 对于Map, reduce, filter这几个Functional方法的使用，可以看Anjana Vakil: Learning Functional Programming with JavaScript - JSUnconf 2016这个视频。YDKJS里面应该也有涉及。

Map

  • 用Object作为Map的缺点在于，Object只能用String作为key，而且这些key不能和Object原型上的方法重名。
  • 不同的引用（不同的对象实例）在Map中是不同的key
  • 可以用for ... of循环来遍历Map，而且遍历的顺序和这个item被加入到Map中的顺序是一致的。

Set

  • Set可以通过在new Set()时传入数组，对数组进行filter转换，达到交集和差集的效果。
